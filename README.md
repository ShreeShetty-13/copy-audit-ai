
#  CopyAudit AI

**CopyAudit AI** is a lightweight, high-conversion landing page designed to help businesses optimize their sales copy using AI-driven benchmarks.

##  Tech Stack
- **Frontend:** React.js (Next.js/Vite)
- **Styling:** Tailwind CSS + Lucide Icons
- **Lead Capture:** Formspree API
- **Payments:** Manual UPI Gateway with UTR Verification

##  Features
- **Split Comparison UI:** Compare "Boring" vs "Growth" copy side-by-side.
- **AI Audit Request:** Secure lead generation form for landing page URLs.
- **Responsive Payment Card:** Integrated UPI QR code and mobile deep-linking.
- **Performance Optimized:** Clean code structure for fast loading and SEO.

##  Getting Started
1. Clone the repo: `git clone [YOUR_URL]`
2. Install dependencies: `npm install`
3. Run locally: `npm run dev`
